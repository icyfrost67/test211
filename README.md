# LoveFlix

A full Netflix-inspired birthday gift website built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Lucide React icons.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Edit your content

- Change password and names: `data/siteConfig.ts`
- Add photos/videos/memories: `data/memories.ts`
- Edit birthday letter: `data/siteConfig.ts`
- Replace local files:
  - `public/images/`
  - `public/videos/`
  - `public/audio/background.mp3`

## Deploy free

1. Push this folder to GitHub.
2. Go to Vercel.
3. Import the GitHub repository.
4. Click Deploy.

For large videos, use short compressed MP4 files or host videos externally and update the media paths.
